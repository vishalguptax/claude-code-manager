# v2.1.0

_Released 2026-06-25_

The sidebar now stays live: skills, commands, hooks, MCP servers, agents, and token usage refresh on file change with no manual reload, resume opens in the IDE you're actually in (Cursor/Windsurf, not VS Code), and the account panel shows your plan with an honest live/idle quota indicator.

## ✨ Features

- **Real-time everything.** Added file watchers for skills, commands, hooks, MCP, and agents so edits under `~/.claude` and the workspace `.claude/` update those tabs live — no Refresh, no Claude session required. ([814aec7](https://github.com/vishalguptax/claude-manager/commit/814aec7f666418669fede8b0b7480ebde8e64da9))
- **IDE-aware resume.** Deep links use the host's own URI scheme (`vscode.env.uriScheme`) instead of a hardcoded `vscode://`, so resuming a session opens in Cursor / Windsurf / VS Code Insiders rather than launching a separate VS Code window. ([814aec7](https://github.com/vishalguptax/claude-manager/commit/814aec7f666418669fede8b0b7480ebde8e64da9))
- **Plan badge.** The Profile badge shows the plan from local credentials — `Pro`, `Max 5x`, `Max 20x` (read from the rate-limit tier). Team is shown bare, since the Standard/Premium seat isn't in any decodable local field. ([814aec7](https://github.com/vishalguptax/claude-manager/commit/814aec7f666418669fede8b0b7480ebde8e64da9))

## 🐛 Bug Fixes

- **Usage no longer freezes mid-session.** Token usage is fingerprinted by transcript file mtime + size (not just directory mtime), so appends to an active session's transcript are detected and the Usage tab updates live. ([814aec7](https://github.com/vishalguptax/claude-manager/commit/814aec7f666418669fede8b0b7480ebde8e64da9))
- **Refresh actually refreshes.** The reload path now clears the model and usage caches and re-pushes quota, so the Refresh button updates the model list, usage, and quota instead of returning stale memoized data. ([814aec7](https://github.com/vishalguptax/claude-manager/commit/814aec7f666418669fede8b0b7480ebde8e64da9))
- **No quota flicker.** Re-reading quota keeps the bars in place instead of tearing down to the spinner. ([814aec7](https://github.com/vishalguptax/claude-manager/commit/814aec7f666418669fede8b0b7480ebde8e64da9))

## 💅 UI / Styling

- Quota: moved the live/idle freshness dot beside Refresh (its tooltip carries "last render Xm ago") and removed the redundant header timestamp and bottom caption. ([814aec7](https://github.com/vishalguptax/claude-manager/commit/814aec7f666418669fede8b0b7480ebde8e64da9))
- Profile: removed the "Open claude.ai" button and the now-dead message path; trimmed the Organization / Plan-since / Credentials rows and aligned the remaining detail into a tidy two-column layout. ([814aec7](https://github.com/vishalguptax/claude-manager/commit/814aec7f666418669fede8b0b7480ebde8e64da9))
