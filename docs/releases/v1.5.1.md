# v1.5.1

_Released 2026-04-21_

Detail-view transcript search feels native now (focus + caret survive every keystroke), plus a new demo GIF headlines the marketplace listing.

## 🐛 Bug Fixes

- sessions: preserve detail search input focus and caret across responses ([35d5d5f](https://github.com/vishalguptax/claude-code-manager/commit/35d5d5fe75621fa61ded8cebd6bbad244c32cfee))

### Highlights

**Detail search no longer feels "stuck"**
The transcript search above the message list re-renders on every debounced response, which previously wiped the input's in-flight value, dropped focus, and reset the caret — so fast typing, mid-word deletes, and list-reload ticks all felt unresponsive. Live value, selection range, and focus are now captured before the `innerHTML` swap and restored after, so typing, deleting, and responses interleave without losing what the user is doing.
