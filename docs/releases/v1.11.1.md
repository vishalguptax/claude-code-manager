# v1.11.1

_Released 2026-05-28_

Marketplace + Open VSX publish of the v1.11.0 changes. The v1.11.0 run created the tag and GitHub Release but skipped the registry publish (publishing credentials were unset), so the Marketplace stayed on 1.10.6. This release carries the same code with the publish step active.

See [v1.11.0](v1.11.0.md) for the full feature notes — in short: the Quota card now reads your 5h / 7d limits **locally** from a statusline tap (no network call, no OAuth token), and a new **Current session** card shows live model, context usage, and session cost.
