# v1.3.1

_Released 2026-04-14_

Layout fixes for action button rows across every detail view, a new Continue toolbar button, and a fix for Restore Workspace opening terminals in separate panels.

## ✨ Features

- **Continue button in the sessions toolbar** — runs `claude --continue` in a terminal at the active workspace cwd, between `+ New` and `Restore Workspace`. Lucide history icon distinguishes it from the per-session Resume button. ([6aa7a99](https://github.com/vishalguptax/claude-code-manager/commit/6aa7a99))

## 🐛 Bug Fixes

- **Restore Workspace race condition** — VS Code registers each new terminal's tab in `tabGroups.all` *asynchronously*, so calling `createTerminal()` in a tight loop made the second and later restored terminals see an empty tab list and open in fresh editor groups instead of stacking. An 80 ms gap between `resumeMultiple` iterations gives VS Code's event loop a tick to register the previous tab. All restored terminals now stack as tabs in a single editor group. ([6aa7a99](https://github.com/vishalguptax/claude-code-manager/commit/6aa7a99))
- **Detail view action buttons wrapped unevenly** — `.d-actions` was a nested flex-wrap structure with a vertical divider before Delete. On narrow panels this produced orphan-row buttons (e.g. Export wrapping alone). Flattened all four detail views (sessions, hooks, mcp, skills) so every button is a direct grid child, and only the primary action spans the full top row. Delete now flows with the rest, no divider needed. ([b89ceba](https://github.com/vishalguptax/claude-code-manager/commit/b89ceba))
- **Toolbar truncating "Restore Workspace"** — the grid track minimum was smaller than the longest label, so the column clipped the text. Bumped from `minmax(95px, 1fr)` to `minmax(135px, 1fr)`; at narrow widths the toolbar now stacks vertically instead of truncating. ([b89ceba](https://github.com/vishalguptax/claude-code-manager/commit/b89ceba))
- **Session detail showing "19714m" for long sessions** — the duration was rendered as raw minutes regardless of magnitude. New `fmtDuration` helper auto-scales the unit so a 13-day session reads as `13d 16h` instead. Pattern: `<1m` / `30m` / `2h 25m` / `13d 16h`. ([b89ceba](https://github.com/vishalguptax/claude-code-manager/commit/b89ceba))
- **Voice toggle thumb blew out when on** — the on-state thumb was pure white against the saturated accent track, dominating the row. Softened to `rgba(255, 255, 255, 0.88)` so it stays readable without screaming. ([b89ceba](https://github.com/vishalguptax/claude-code-manager/commit/b89ceba))
- **Import / Export icons inverted from intuition** — used `download` (arrow IN) for Export and `upload` (arrow OUT) for Import, which matched web download/upload semantics but conflicted with intuition for local file actions. Swapped: Export now uses `upload` (arrow OUT), Import now uses `download` (arrow IN). ([b89ceba](https://github.com/vishalguptax/claude-code-manager/commit/b89ceba), [6aa7a99](https://github.com/vishalguptax/claude-code-manager/commit/6aa7a99))

## 💅 UI / Styling

- All action button rows across every feature (`.d-actions`, `.actions-bar`, `.acct-actions`, `.agent-detail-actions`, `.cmd-detail-actions`) converted from `flex-wrap` to `display: grid` with `repeat(auto-fit, minmax(N, 1fr))`. Buttons distribute evenly at every width and reflow cleanly on narrow panels. ([b89ceba](https://github.com/vishalguptax/claude-code-manager/commit/b89ceba))
- `min-width: 0` on `.d-actions` buttons so 1fr columns become truly equal width regardless of label length — default `min-width: auto` pinned columns to content size, breaking equal-expand. ([b89ceba](https://github.com/vishalguptax/claude-code-manager/commit/b89ceba))
- `justify-content: center` on stretched action buttons so icon + label stay centered when the grid expands a button wider than its natural content. ([b89ceba](https://github.com/vishalguptax/claude-code-manager/commit/b89ceba))
- Removed the orphaned `.mcp-detail-actions` CSS rule (no HTML reference) and dead `.d-actions-secondary` / `.d-actions-divider` rules. ([b89ceba](https://github.com/vishalguptax/claude-code-manager/commit/b89ceba))
