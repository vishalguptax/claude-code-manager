# v2.0.0 (work in progress)

Per-feature migration notes for the Preact revamp. Assembled into the final
CHANGELOG during F3 integration.

## mcp

- Migrated the MCP servers tab to Preact with `@preact/signals` state.
- Added valibot validation on every postMessage, on both the webview send
  path (`createMcpApi`) and the host receive path (`handleMcpMessage`).
- Extracted host dispatch into `features/mcp/messageHandlers.ts` (validated,
  self-contained, no cross-feature imports).
- Large server lists (>50) now render through the shared windowed
  `<VirtualList />`; smaller lists render grouped by scope as before.
