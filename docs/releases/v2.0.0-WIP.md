# v2.0.0 (work in progress)

Per-feature migration notes for the v2 Preact revamp. Each F2 session appends
its own section; the integration session assembles the final CHANGELOG.

## commands

- Migrated the commands webview from vanilla DOM to Preact with `@preact/signals`.
- Inbound messages from the webview are now validated against the shared valibot protocol on the host (`dispatchCommandsMessage`), and host messages are routed into feature signals via the shared message bus.
- List view groups commands by scope, debounces search, and windows the rows with `<VirtualList />` once the list grows past 50 entries.
- Extracted host dispatch into `messageHandlers.ts` with injected VS Code capabilities so it is unit-testable without the extension host.
