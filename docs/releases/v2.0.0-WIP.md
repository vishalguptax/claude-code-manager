# v2.0.0 (work in progress)

Per-feature changelog entries assembled during the F2 migration. Finalized by
the F3 integration session.

## agents

- Migrated the agents webview from vanilla DOM to Preact with `@preact/signals`.
- Inbound `agents`/`error` messages now flow through the shared message bus and
  are validated by valibot (`parseMessage`); outbound `getAgents`/`openAgentFile`
  go through the typed `useApi()` bridge.
- Large agent lists (> 50 entries) render through the shared windowed
  `VirtualList`; smaller lists keep scope-grouped sections (Project / Global /
  Plugin). No user-visible behavior change.
- Feature styles consolidated into `src/styles/agents.css` (no inline styles, no
  `innerHTML`).
