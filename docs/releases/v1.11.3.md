# v1.11.3

_Released 2026-05-28_

Small additive polish: the MCP tab now badges connectors that Claude Code has flagged as needing (re-)authentication, so you can spot a stale Gmail / Drive / Calendar login without opening Claude.

## ✨ Features

- mcp: surface MCP connectors that need re-auth as a banner at the top of the MCP tab, sourced from Claude Code's own `~/.claude/mcp-needs-auth-cache.json`. No network, no polling — strictly reads what Claude already wrote.

## Notes

- Pure read of Claude Code's auth-needs cache (keys are connector display names like "claude.ai Gmail"). Empty/missing file → no banner.
- Piggybacks the existing `mcpServers` message — no protocol or schema changes.
