# v2.4.0

_Released 2026-07-04_

Polish and validation pass over the new MCP / agent / hook management UI: inline form validation, a cleaner MCP detail view, and a sidebar-wide layout/consistency cleanup.

## ✨ Features

- **Full inline form validation** for the agent, MCP, and hook forms — name format + duplicate-name checks (per scope), URL-scheme validation per MCP transport, `KEY=value` env/header line checks that name the offending line, and positive-integer hook timeouts. Every rule shows an inline hint and gates Save; the host writers stay as the backstop. ([c8945ad](https://github.com/vishalguptax/claude-code-manager/commit/c8945ad9bd5499db8cce3239f7800dccfdd9a5f2))

## 🐛 Bug Fixes

- **MCP detail view** — folded the 8-button action wall into primary actions (Edit / Enable-Disable / Delete) plus a scroll-safe "More" overflow menu; gave the user-scope and plugin notes their panel inset; fixed the reveal-values toggle spacing. Also swept panel insets, unified empty states, closed dropdown popups on scroll, and fixed an `ErrorBanner` key collision. ([d2d5583](https://github.com/vishalguptax/claude-code-manager/commit/d2d55838447420222628daa9aaa283b06ba6dd9e))
- **Forms & lists** — create/edit forms now render as full inline panels instead of cramped modals (which stacked their own dropdown popups behind the backdrop); compact MCP action row; corrected banner padding; hook list rows read as text instead of disabled input boxes; removed a global `.back-btn` rule that was clobbering the hooks/commands/skills back buttons. ([9e2f712](https://github.com/vishalguptax/claude-code-manager/commit/9e2f712fd855025acbf68f0caa8ed5ece483ae3e))

## ⚡ Performance

- Self-host fonts and drop the render-blocking Google Fonts request on the marketing site. ([674575e](https://github.com/vishalguptax/claude-code-manager/commit/674575e024bf3036ac14df0e9b7ab430474d83ab))
