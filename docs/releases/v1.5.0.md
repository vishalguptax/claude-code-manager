### Highlights

**Multi-account switcher (Accounts)**
Snapshot any Claude login and swap between accounts without re-running `/logout` + `/login`. Native QuickPick opened from the Profile button, the clickable avatar, or the "Claude Manager: Switch Account" command-palette entry. One-time security disclaimer modal on first save. Active profile detection via SHA-256 of credentials. Atomic switch (tmp + rename).

**Quota card (opt-in)**
Current subscription utilization — 5-hour / 7-day windows, per-model breakdown, pay-as-you-go overflow. Single OAuth call to `api.anthropic.com/api/oauth/usage` using your own token (read from `~/.claude/.credentials.json`; never transmitted). Auto-fetch after first opt-in, 5-minute TTL cache, instant paint across tab switches, header refresh button.

**Transcript search + detail enrichment**
Sticky search bar above the message list searches every message (content + extended thinking + tool calls) across the whole session. Live match count + `<mark>` highlights. Per-message token stamp (input / output / cache). Tool-use rows ("Read src/foo.ts", "Bash npm test"). Collapsible extended-thinking blocks. Per-message Copy + Ask Again buttons. Latest mode now shows newest first.

**Claude Code extension integration**
- New `claudeManager.sessions.resumeIn` setting routes Resume / New / Continue to the extension chat tab or a terminal. `auto` follows the session's origin.
- Sessions started inside the extension now appear in the list even though the extension skips `history.jsonl`.
- "Launch in Chat" buttons on slash commands + skills, "Ask Again" on user prompts in session detail, "Open & Chat" for cross-project navigation.
- Cross-workspace routing opens the target project then fires the URI after a short activation delay.

**Skills**
Recursive subdirectory walk (`team/lint/SKILL.md` etc.) with stable grouping + collision-safe IDs. YAML block-scalar support in SKILL.md frontmatter (`description: >-` now reads the folded prose instead of the literal marker).

**Usage stats fixes**
Active days now period-scoped. Current streak anchors on the latest recorded date (not wall-clock). UTC-safe streak math. Windows path-casing dedupe. Orphan-session discovery with slug-decode fallback when transcript `cwd` is missing.
