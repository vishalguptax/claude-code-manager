# v2.2.0

_Released 2026-06-26_

A responsiveness and honesty release: lists scroll smoothly with a stable scrollbar, timestamps and quota tick live, settings react instantly, the MCP auth badge and account-switched quota behave correctly, the packaged extension is far smaller, and the privacy docs now state the truth — zero network calls.

## ✨ Features

- **Smooth variable-height lists, a live clock, and instant dropdowns.** The virtualized lists now measure real row heights, so mixed-height rows (short group headers next to taller items) stop drifting — scrolling is smooth and the scrollbar thumb no longer shrinks as you scroll. A shared clock keeps relative timestamps and the quota "resets in" countdown live without per-view timers, and model/mode/effort dropdowns reflect your choice instantly instead of waiting on the host round-trip. ([365bca1](https://github.com/vishalguptax/claude-manager/commit/365bca1269b848e809ab7b2f0e516b89e4114d78))

## 🐛 Bug Fixes

- **MCP auth badge no longer vanishes, and quota is honest across account switches.** Refresh and live `.mcp.json` edits kept the auth-health badge. After an account switch the quota card no longer shows the previous account's numbers — it waits for a fresh render — and the live/idle dot now flips to idle on its own instead of blinking forever. The model dropdown also dropped a duplicate "Opus 4.8" and stopped claiming a specific Default model it can't actually read. ([c3795f4](https://github.com/vishalguptax/claude-manager/commit/c3795f44dfc06c0b76d871e51e02e30aa8579a4d))

## 📦 Build / CI

- **Much smaller package + correct site counters.** The `.vsix` was shipping 160+ unrelated repo files (agent tooling, the marketing site, internal docs); excluding them drops the package from 225 files to 36 (~679 KB). The webview build now purges stale content-hashed chunks before each run, and the site's live star/download counters actually update again (the bake was matching the wrong element tag). ([9d1ffe2](https://github.com/vishalguptax/claude-manager/commit/9d1ffe274edbe7e9d0331c0159d76c486ccbcde6))

## 📚 Documentation

- **Accurate privacy claims and more human copy.** The README and site described removed v1 behavior (a quota network call); quota is now read locally from Claude Code's statusline, so the docs state the accurate, stronger claim: zero network calls, ever. The prose was rewritten to read less machine-generated, the sponsor-slot link now opens an email inquiry, and an unverifiable structured-data rating was removed. ([df00e26](https://github.com/vishalguptax/claude-manager/commit/df00e26283943ced0cd7289b3e9bfa0eaadb106e))
- Use the canonical `claude-manager` repo slug in the release notes. ([cac475b](https://github.com/vishalguptax/claude-manager/commit/cac475baf242ba4ef57b1e3f4f80e8ce9a679c2c))
