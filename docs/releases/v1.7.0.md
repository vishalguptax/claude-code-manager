# v1.7.0

_Released 2026-04-26_

Backlog batch — eight new capabilities land in one release: full hook CRUD, bulk session ops, cost estimates, settings rollback, self-diagnostic, reload command, marketplace links, and tighter usage stats.

## ✨ Features

- inline enable/disable, edit, add, delete on the Hooks tab — no more hand-editing settings.json ([153adb8](https://github.com/vishalguptax/claude-code-manager/commit/153adb813d1189f2bb0b8f969239f998e03b0336))
- multi-select toolbar in Sessions — Pin/Unpin/Export-as-zip/Delete with shift-click range and Ctrl/Cmd+A select-visible ([73f470f](https://github.com/vishalguptax/claude-code-manager/commit/73f470f91e4dbf96ba2835490449f5fccc9d538e))
- self-diagnostic command — opens a markdown report with CLI / file / hook / version checks ([369696e](https://github.com/vishalguptax/claude-code-manager/commit/369696e4585bd2041c8e44bbf18ac1680d32f86d))
- settings snapshot history — every settings.json mutation auto-snapshots and Config tab gains a Restore list ([14d8cc8](https://github.com/vishalguptax/claude-code-manager/commit/14d8cc852a0b92d57b7fbc6b8c3f68103b754f7b))
- cost estimate per model + total in Account → Usage, with a date-stamped price snapshot ([961be78](https://github.com/vishalguptax/claude-code-manager/commit/961be78c9175968756374f0eb7ca5ee147eba2a0))
- marketplace external links on Skills + MCP tabs — configurable via `claudeManager.marketplaceSkillsUrl` / `claudeManager.marketplaceMcpUrl` ([cdc2f30](https://github.com/vishalguptax/claude-code-manager/commit/cdc2f305b95c89617441512802852d1676dd6eb5))
- reload command + toolbar button — re-parse every tab without recreating the webview (Ctrl/Cmd+Alt+R) ([a08609a](https://github.com/vishalguptax/claude-code-manager/commit/a08609a3aca43d9adc618a6eccbba19443d7a9a2))
