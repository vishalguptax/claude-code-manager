# Changelog

## [0.1.0] - 2026-04-08

### Added
- Session Manager: browse, search, pin, delete, resume Claude Code sessions
- Skills Manager: view global and project-level skills with SKILL.md content
- Commands Viewer: browse slash commands from ~/.claude/commands/
- Hooks Manager: view hooks configured in settings.json
- MCP Servers: view .mcp.json server configurations with type badges
- Agents: browse .claude/agents/ with model badges
- Tab navigation between all 6 features
- Project filter with custom dropdown
- Date filter chips (Today/Week/Month/All)
- Search with debounce
- Right-click context menu (pin, fork, copy, delete)
- Branch detection with switch prompt on resume
- Cross-project session resume
- Lucide icons throughout
- VS Code native theme integration
- 76 unit tests
